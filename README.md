# Algorithm of the Day (AoTD)

A personal repository for daily algorithm practice. The goal is to solve one algorithm problem every day to build consistency, improve problem-solving skills, and explore different data structures and approaches.

## Repository Structure
Each algorithm solution is stored as a separate file in the root directory, named after the problem (e.g., `weird-algorithm.js`). Files are written in JavaScript (Node.js) by default, with support for other languages added as needed.

## Getting Started
### Prerequisites
- [Node.js](https://nodejs.org/) installed to run JavaScript files.

### Run a Solution
```bash
node [filename].js
```
Example: `node weird-algorithm.js`

## Problem Sources
Common platforms used for daily problems:
- [CSES Problem Set](https://cses.fi/problemset/)
- [LeetCode](https://leetcode.com/)
- [Codeforces](https://codeforces.com/)
- [HackerRank](https://www.hackerrank.com/)

## Notes
- Solutions include problem context, implementation, and basic test cases where applicable.
- Adjust the structure (e.g., date-based folders, language-specific directories) as the repo grows.
